# vue-scaffold-map

> vue项目脚手架

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9000
npm run dev

# build for production with minification
npm run build
```
## 第三方 依赖

### UI库：
* iview

### 插件：
* echarts
* tinymce: 富文本编辑器
* jquery
* fuse: 模糊搜索
* esri-loader: arcgis

### 数据模拟：
* mock.js

### css预编译器：
* less
